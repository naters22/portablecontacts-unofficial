PortableContacts for {{ domain }}

This is an unofficial PortableContacts server for {{ domain }}. It serves these endpoints:

  • https://{{ host }}/.well-known/host-meta
  • https://{{ host }}/user?uri={uri}

The user URI is acct:username@{{ domain }}, where username is a username or user id. See the PortableContacts docs for usage details.
